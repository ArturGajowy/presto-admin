***********************
Multi Node Installation
***********************

The following describes installing Presto on a cluster of nodes via the ``presto-admin`` software. This is an alternative to the installation steps described at `prestodb.io <https://prestodb.io/docs/current/installation.html>`_. Using the ``presto-admin`` tool is the simplest and preferred method for installing and managing a Presto cluster.
